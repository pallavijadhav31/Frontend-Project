1.Overview of Project 1.1HTML &CSS Project on Airline reservation system This project airline reservation system has been developed on HTML &CSS. The main aim of developing this project as to create a static website for the airlines, such as about the airlines, airline schedule, airline booking, contact details, address, images and etc. This project is in HTML&CSS so it’s a static website and dosen’t perform any type of dynamic database operation.

1.2 Pages of the Project Airline System • Home Page: This project contents the basic information about the airlines system. • About Page: It contain about the airline system information and their working. • Register Page: Here users can easily register on the airlines website and confirmation message will be displayed after registration. • Contact Page: User can submit their contact details on the airlines website. • Login: User can log in in the airlines website. • Forget Password: User can create a new password.


1.3 Technology Used in the project Airline System • HTML: Page layout has been designed in HTML. • CSS: CSS has been used for all the designing part.


1.4 Supported Operating System We can configure this project on the following the operating system: • Windows: This project can easily be configured on windows operating system.
